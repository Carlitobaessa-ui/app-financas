Reconciliar Saldos
O saldo de cada conta deveria ser saldo inicial + créditos − débitos dos lançamentos. Quando isso não bate — geralmente após importar extrato ou restaurar um backup antigo — você escolhe qual é a verdade:
Os lançamentos estão certos
Recalcula o saldo = inicial + lançamentos. Ideal quando você confia no extrato.
O saldo atual está certo
Mantém o saldo e corrige o saldo inicial. Ideal quando o saldo mostrado é o real e os lançamentos são histórico.
🛡 Um snapshot de backup é criado automaticamente antes de qualquer ajuste, então você pode desfazer pela Central de Backup.

Configurações
Chave de API — Agentes de IA
Os três assistentes usam a API da Anthropic. Para funcionarem localmente no navegador (fora do Claude.ai), cole aqui sua chave pessoal. Ela fica salva apenas neste navegador (localStorage).
⚠ Segurança: a Anthropic desaconselha usar a chave direto no navegador. Qualquer pessoa com acesso a este arquivo ou computador pode lê-la. Não compartilhe o arquivo após colar a chave. Se ela vazar, revogue-a no console da Anthropic. O uso da API é cobrado por tokens.
Obtenha sua chave em console.anthropic.com → API Keys.
Segurança dos Dados
Central de Backup
Duas camadas de proteção: snapshots internos automáticos (até 3/dia, rede de segurança contra exclusão ou importação errada) e exportação em arquivo .json — o backup que sai do app e sobrevive à limpeza do navegador ou perda do aparelho.
